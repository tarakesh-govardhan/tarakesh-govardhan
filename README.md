# About Me

My interest in cybersecurity began after watching the documentary **“We Are Legion: The Story of the Hacktivists”**, which sparked my curiosity about hackers, security, and how systems can be broken.

Since then I have spent years exploring Linux environments and security tools, starting with distributions like **Ubuntu, BackTrack, and Linux Mint** during school.

During my Computer Science degree, I completed my final-year project on **Intrusion Detection Systems using Machine Learning**, which deepened my interest in cybersecurity.

After a short career detour into financial auditing, I made the decision to fully commit to cybersecurity and transitioned into a **Cybersecurity Analyst role**, where I continue building practical security and offensive testing skills.
